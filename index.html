공지사항

이 도구에 대해

코코포리아에서 내보낸 HTML 로그를 꾸며서 백업하는 도구입니다. 모든 처리는 브라우저 안에서만 이루어지며, 로그가 서버로 전송되거나 저장되지 않습니다.

시간순 통합 정렬 기능

탭별로 따로 출력한 HTML 파일 여러 개를 올린 뒤, 시간 수집 도우미 코드로 수집한 시간 데이터를 붙여넣으면 탭 내용이 실제 발언 시각 순서로 하나로 합쳐집니다.

주의

코코포리아 측 업데이트로 도우미 코드가 동작하지 않게 될 수 있습니다. 동작하지 않으면 알려주세요.
도움말

사용 방법

  1. 코코포리아에서 로그를 내보냅니다. (전체로그 또는 탭별 출력)
  2. 내보낸 HTML 파일을 업로드하면 탭·캐릭터가 자동 감지됩니다.
  3. 도우미 코드로 캐릭터 이미지와 시간 데이터를 불러옵니다.
  4. 스타일 조정 후 백업합니다.

캐릭터 편집

상단 캐릭터 편집 탭에서 캐릭터별 이름 색상·기본 이미지·표정을 관리합니다. 메시지에 적힌 @라벨에 따라 표정이 자동 매칭됩니다.

탭 · 캐릭터 필터

칩을 클릭해 켜고 끄면 편집·미리보기에 즉시 반영됩니다. 색상이 지정되지 않은 캐릭터는 흐릿한 칩으로 표시되며, 기본색 캐릭터 숨기기로 일괄 제외할 수 있습니다.

범위 선택 삭제

로그 편집 탭에서 범위 선택 삭제를 켠 뒤 시작 메시지와 종료 메시지를 차례로 클릭하고 확정을 누르면 그 사이가 일괄 삭제됩니다. Ctrl+Z로 복구할 수 있습니다.

마크다운 자동 변환

*기울임* · **볼드** · ***볼드+기울임*** 이 출력 시 자동 적용됩니다.

단축키

Ctrl+Z 되돌리기 · Ctrl+F 검색 · Ctrl+S HTML 저장 · Ctrl+P PDF 저장
시간 수집 사용법
코코포리아 내보내기 HTML에는 시간 정보가 없어서, 룸에서 도우미 코드로 원본 시각을 수집해야 탭들을 정확한 순서로 섞을 수 있습니다.

처음 한 번만 설정

  1. 시간 수집 도우미 코드 복사 버튼을 누릅니다.
  2. 브라우저에서 새 북마크를 만들고, URL 칸에 복사한 코드를 그대로 붙여넣어 저장합니다.

수집 (방마다 한 번)

  1. 코코포리아 룸에서 북마크를 클릭하면 좌측 하단에 수집 패널이 나타납니다.
  2. 채팅 탭을 하나 열고 [자동 스크롤]을 누르면 과거 메시지를 자동으로 로드하며 수집합니다. "끝까지 로드된 것 같습니다"가 뜨면 다음 탭을 열고 다시 누릅니다.
  3. 탭별 누적 개수가 로그 분량과 맞으면 완료·복사를 누릅니다.
  4. 이 사이트의 시간 수집 칸에 붙여넣고 불러오기를 누른 뒤, 시간순 통합 정렬을 켭니다.

참고

수집 데이터에는 각 메시지의 정확한 시각(밀리초)과 인장 이미지 URL이 포함되어, 정렬과 동시에 캐릭터 이미지도 자동 등록됩니다. 볼 권한이 없는 탭은 수집되지 않으므로, 해당 탭은 권한 있는 사람(예: GM)이 같은 방법으로 수집해 전달해주면 합칠 수 있습니다. 이 코드는 화면에 로드된 데이터만 읽으며 외부로 전송하지 않습니다.

시간 수집 사용법
코코포리아 내보내기 HTML에는 시간 정보가 들어있지 않기 때문에, 룸 화면에서 시간을 직접 수집해야 탭별 파일을 시간순으로 섞을 수 있습니다.

처음 한 번만 설정

  1. 메시지 수집 코드 복사 버튼을 누릅니다. (이 코드 하나로 시간과 표정 매칭을 함께 처리합니다)
  2. 브라우저에서 새 북마크를 만들고, URL 칸에 복사한 코드를 그대로 붙여넣어 저장합니다. (이름 자유, 예: 시간수집)
  3. 북마크바가 안 보이면 Ctrl+Shift+B로 켭니다.

탭마다 반복

  1. 코코포리아 룸에서 수집할 채팅 탭을 엽니다.
  2. 북마크를 클릭하면 좌측 하단에 수집 패널이 나타납니다.
  3. 채팅창을 맨 아래에서 위로 끝까지 스크롤하면 보이는 메시지가 시간과 함께 누적됩니다.
  4. 패널에 "방식: 내부 데이터"가 표시되는지 확인한 뒤, 완료·복사를 누르고, 이 사이트의 시간 수집 입력칸에 붙여넣어 불러오기를 누릅니다.
  5. 다른 탭도 같은 과정을 반복합니다. (여러 번 붙여넣으면 자동 누적됩니다)

정렬

수집이 끝나면 시간순 통합 정렬을 켜세요. 업로드한 탭별 파일의 메시지가 시간 데이터와 자동 매칭되어 하나로 합쳐집니다. 시간이 매칭되지 않은 메시지는 앞뒤 메시지 사이 순서를 유지한 채 끼워 넣어집니다.

주의

이 코드는 화면에 표시된 내용만 읽으며, 외부로 데이터를 전송하거나 코코포리아 서버에 접근하지 않습니다. 수집 후 로그를 수정하면 매칭률이 떨어질 수 있습니다.
이미지 URL 일괄 얻는 방법
코코포리아 화면에 표시된 이미지 URL만 읽어오는 도우미 코드입니다. 외부로 데이터를 전송하지 않습니다.

캐릭터별 이미지 URL 가져오기

  1. 캐릭터 이미지 코드를 복사해 북마크 URL로 저장합니다.
  2. 코코포리아에서 캐릭터 편집창을 연 상태로 북마크를 클릭합니다.
  3. 표시된 결과를 복사해 이미지 URL 불러오기 칸에 붙여넣고 불러오기를 누릅니다.
  4. 캐릭터마다 반복합니다.

메시지별 표정 자동 매칭

시간 수집 섹션의 메시지 수집 코드를 북마크로 저장한 뒤 룸 채팅 화면에서 클릭하면 수집 패널이 나타납니다. 채팅창을 스크롤해 메시지를 누적한 뒤 완료를 누르고, 결과를 같은 칸에 붙여넣으면 @라벨에 따라 표정이 자동 매칭됩니다. 시간과 표정 매칭이 한 번의 수집으로 함께 처리됩니다.
